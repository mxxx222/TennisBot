# Archive Directory

This directory contains all files that are not related to the ITF tennis pipeline.

## Structure

- `football/` - Football/soccer related files
- `betfury/` - Betfury betting platform files
- `crypto/` - Cryptocurrency trading files
- `venice_ai/` - Venice AI integration files
- `mojo/` - Mojo performance layer files
- `betting_intelligence/` - General betting intelligence systems
- `social_intelligence/` - Reddit/Discord/Twitter intelligence
- `web_dashboard/` - Web dashboard and Vercel files
- `educational/` - Educational and testing systems
- `general_tennis/` - General tennis (non-ITF) files
- `svd/` - SVD system files
- `multi_sport/` - Multi-sport and unified systems
- `weather/` - Weather integration files
- `notion_general/` - General Notion integration (non-ITF)
- `tests/` - Test files (non-ITF)
- `telegram_monitoring/` - Telegram and monitoring (non-ITF)
- `database/` - Database and relational DB files (non-ITF)
- `n8n_betflow/` - N8N workflows and Betflow Pro
- `setup_deployment/` - Setup and deployment scripts (non-ITF)
- `docs/` - General documentation

## Note

These files have been archived to focus the repository on ITF tennis pipeline functionality.
All files are preserved and can be restored if needed.

Total archived: ~28,000 files, ~521MB
